# Workflow: Refactor

## Required Sequence
1. Define invariants and boundaries. Use `ai/skills/02_root_cause.md` for system mapping.
2. Add characterization tests if behavior is unclear. Use `ai/skills/03_write_failing_test.md`.
3. Apply a minimal refactor with small diffs. Use `ai/skills/04_minimal_fix.md`.
4. Verify behavior and add minimal observability if needed. Use `ai/skills/05_regression_plus_observability.md`.
5. Produce a PR-ready summary. Use `ai/skills/06_pr_ready_summary.md`.

## Required Artifacts
- Change Plan
- PR Summary
