# General Workflow

## Process
1. Select the workflow that matches the task.
2. Produce a Change Plan before modifying code.
3. Execute the workflow steps in order.
4. Validate with tests or documented evidence.
5. Produce a PR Summary.

## Required Deliverables
- Change Plan using `ai/templates/change_plan.md`.
- PR Summary using `ai/templates/pr_summary.md`.

## Evidence
- Every claim must be backed by a test result, repro log, or file reference.
