# Workflow: Feature

## Required Sequence
1. Define acceptance criteria and scope. Use `ai/skills/01_repro_min.md` for validation inputs.
2. Define tests that prove the criteria. Use `ai/skills/03_write_failing_test.md`.
3. Implement minimal functionality to satisfy criteria. Use `ai/skills/04_minimal_fix.md`.
4. Verify behavior and add minimal observability if needed. Use `ai/skills/05_regression_plus_observability.md`.
5. Produce a PR-ready summary. Use `ai/skills/06_pr_ready_summary.md`.

## Required Artifacts
- Change Plan
- PR Summary
