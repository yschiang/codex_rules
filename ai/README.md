# AI Playbook Assets

This folder contains agent-focused guidance, workflows, skills, and templates used by the playbook.

## Contents
- `guidelines.md`: non-negotiable rules and how they map to `clinerules/`.
- `workflow.md`: general process and required deliverables.
- `skills/`: step-by-step skills aligned to workflows.
- `templates/`: artifact templates for change plans and PR summaries.
