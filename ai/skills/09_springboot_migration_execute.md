# Skill: SpringBoot Migration Execution

## Purpose
Implement the migration steps with minimal, auditable changes.

## Inputs
- Change Plan artifact
- Compatibility Gap List
- Existing tests and build commands

## Steps
1. Apply changes step-by-step, keeping diffs minimal.
2. Update dependency management and configuration as planned.
3. Run targeted tests or startup checks after each step.
4. Record what changed and why.

## Outputs
- Migration Change Log artifact with files changed and rationale.
- Build Or Startup Evidence artifact with commands and results.
