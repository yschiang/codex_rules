# Skill: PR Ready Summary

## Purpose
Produce a complete PR summary suitable for review and release decisions.

## Inputs
- Change Plan
- Fix Notes and Regression Coverage
- Test results
- Risk assessment

## Steps
1. Summarize the issue or goal and root cause or design intent.
2. List the changes and any tradeoffs.
3. Record tests with commands and results.
4. Document risks and rollback.

## Outputs
- PR Summary artifact using `ai/templates/pr_summary.md`.
