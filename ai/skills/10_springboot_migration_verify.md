# Skill: SpringBoot Migration Verification

## Purpose
Verify application health, compatibility, and regression coverage after migration.

## Inputs
- Migration Change Log
- Tests and startup commands
- Observability requirements

## Steps
1. Run regression tests or targeted suites relevant to the migration.
2. Verify application startup and key endpoints or health checks.
3. Validate configuration and dependency alignment.
4. Note any new warnings or behavior changes.

## Outputs
- Verification Report artifact with tests, startup checks, and findings.
- Risk Notes artifact if any regressions or compatibility issues remain.
